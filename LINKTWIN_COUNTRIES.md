# Linktw.in Geo-Targeting Countries Reference

Complete list of all 252 countries/territories supported by the Linktw.in geo-targeting API.

> **Note:** The original Linktw.in documentation says "249 countries" but the actual dropdown
> contains 252 distinct entries (including territories like Hong Kong, Puerto Rico, etc.).

## How to Use This Reference

The script `bulk_update_links.py` only includes a subset of these countries in its
`COUNTRIES` list. To add a new country:

1. Find the **exact name** in the table below (spelling and punctuation must match)
2. Add it to the `COUNTRIES` list in `bulk_update_links.py` in **alphabetical order**
3. Update the country count in the docstring (line 6) and comment (line 39)
